# Tech Stack

## Core
- **Next.js 15** - React framework with App Router
- **TypeScript 5.6** - Type-safe development
- **Bun 1.0+** - Fast package manager and runtime

## Backend
- **Supabase** - PostgreSQL database
- **Drizzle ORM 0.44** - Type-safe SQL queries
- **NextAuth v5** - Authentication
- **tRPC 11.7** - Type-safe APIs

## Frontend
- **React 18** - UI library
- **Tailwind CSS 3.4** - Styling
- **Radix UI** - Accessible components
- **Framer Motion** - Animations
- **TanStack Query** - Server state
- **Zustand** - Client state

## AI & APIs
- **Vercel AI SDK** - AI streaming
- **Google Gemini** - Primary AI provider
- **OpenAI** - Fallback AI provider

## Testing
- **Vitest 4.0** - Unit tests
- **Playwright 1.48** - E2E tests
- **Testing Library** - Component tests

## DevOps
- **Biome 2.3** - Linting & formatting
- **GitHub Actions** - CI/CD
- **Vercel** - Hosting

## Why This Stack?

### Bun over npm/yarn
- 10x faster installs
- Native TypeScript support
- Better DX

### Drizzle over Prisma
- Smaller bundle size
- Better performance
- SQL-first approach

### NextAuth v5
- Future-proof
- Better edge support
- Simpler API

### tRPC over REST
- End-to-end type safety
- No code generation
- Better DX

### Biome over ESLint + Prettier
- 100x faster
- Single tool
- Zero config
